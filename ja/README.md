# コードの壁 · The Code Barrier — 日本語版

> **蒸留・コンピュータ操作エージェント・キャッシュ——自分だけのローカル知能を「作り、使い、養う」。**

繁体中文の正典を底本とした完全日本語版。第 1〜11 章（三つの楽章：**作る**＝第 1〜6 章／**使う**＝第 7〜9 章／**養う**＝第 10〜11 章）＋ 付録 A〜R を収録。

## 読み方・ビルド

純 Python・サードパーティ依存なし：

```bash
python _build.py   # 合本 .md ＋ サイドバー目次つき .html（PDF は HTML から印刷）
```

| 成果物 | 内容 |
|--------|------|
| `TheCodeBarrier_full.md` | 全章＋全付録の合本 Markdown |
| `TheCodeBarrier.html` | サイドバー目次つき HTML（PDF はこの HTML から印刷） |

## 真偽の規律

実在の工学と未検証の伝聞が併存する。未検証の製品名・仕様・スコア・遅延はすべて「⚠️ 真偽メモ」で印をつけた——**方向は信じ、細部は疑え**。未認可の能力抽出・アクセス制御の回避・安全アライメントの解除に触れる内容は**リスク開示であって操作助言ではない**（第 6 章・付録 E を参照）。

> 一部の付録（I/J/M）には、底本の原資料が途切れていた箇所がある。該当欄は **表題からの推論で再構成**し、`〔⚠️ 原資料が途切れ・表題からの推論で再構成〕` と明示した。
